Synopsis

This web page calculates the best way to combine bobbins of spun yarn to get an even result.

What it does

Calculate the average weight of each bobbin's yarn (you give the net weight and length) and then tests all the combinations to find the one that matches best. That means when you ply the yarn, the total thickness should be consistent.

How to use it

There are 3 panels in the screen, left to right.

First enter the number of bobbins and the number of ply (usually 2-4) in the left panel. Notice that the number of permutations appears lower down. This is how many different assortments the program will have to test.

Next enter all your measurements in the middle panel. When you are done, the button Find the best combination will light up on the left panel. Press it.

When you press Find the best combination, the computer goes into high gear! Current mode is Calculating. The progress is shown in "Found so far" and provisional results shown in the right panel.

When the program is complete, Current mode will be Complete and the result in the right panel.
